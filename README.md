# mat-mat-ops
CUDA implementation of some Matrix-Matrix Operations.
This repo has both custom CPU and CUDA kernels.

## Requirements
CUDA Toolkit 12.4
PyTorch 2.4+

##To build:
```
pip install .
```

##To test:
the interactive option : `test/test.ipynb`
or
```
python test/test_extension.py
```
